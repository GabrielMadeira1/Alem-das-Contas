## Público-alvo

Este curso foi desenvolvido para qualquer pessoa que queira aprender matemática desde o início, mesmo sem conhecimento prévio.

É indicado para:

- Estudantes que desejam construir uma base sólida em matemática.
- Alunos do Ensino Fundamental e Ensino Médio.
- Pessoas que têm dificuldade na disciplina e querem aprender de forma clara e progressiva.
- Quem está se preparando para ENEM, vestibulares e concursos e deseja fortalecer seus fundamentos matemáticos.

## Objetivo

Construir uma base sólida em matemática, desenvolvendo compreensão em vez de memorização.
